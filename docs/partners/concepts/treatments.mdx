A **Treatment** is a service offered by a venue. Each venue has its own
catalogue. Treatments can have **variants** (e.g. a "Swedish massage" with
30/60/90-minute options, or single / couple variants).

<Note>
  Prices are **decimal amounts in the venue's currency** (e.g. `120.00` for
  120 €), not integer cents. Pair each price with the treatment's `currency`
  field.
</Note>

## The Treatment object

<ResponseField name="id" type="string">
  Stable, opaque identifier (UUID).
</ResponseField>

<ResponseField name="slug" type="string | null">
  Human-readable identifier for the treatment.
</ResponseField>

<ResponseField name="name" type="string">
  Display name in French.
</ResponseField>

<ResponseField name="name_en" type="string | null">
  Display name in English.
</ResponseField>

<ResponseField name="description" type="string | null">
  Description in French.
</ResponseField>

<ResponseField name="description_en" type="string | null">
  Description in English.
</ResponseField>

<ResponseField name="category" type="string | null">
  High-level grouping (e.g. `massage`, `facial`, `body`, `ritual`).
</ResponseField>

<ResponseField name="service_for" type="string | null">
  Intended audience: `women`, `men`, `all`, or `couple`.
</ResponseField>

<ResponseField name="duration" type="integer">
  Duration in minutes for the default variant.
</ResponseField>

<ResponseField name="price" type="number | null">
  Public price for the default variant, as a decimal amount in `currency`.
  `null` when `price_on_request` is `true`.
</ResponseField>

<ResponseField name="price_on_request" type="boolean">
  If `true`, price is not public and the customer must request a quote.
</ResponseField>

<ResponseField name="currency" type="string | null">
  ISO 4217 currency code for `price` (e.g. `EUR`).
</ResponseField>

<ResponseField name="lead_time" type="integer | null">
  Minimum number of minutes between "now" and the earliest bookable start.
  Used to prevent last-minute bookings.
</ResponseField>

<ResponseField name="image" type="string | null">
  Illustration URL.
</ResponseField>

<ResponseField name="sort_order" type="integer | null">
  Ordering hint within the catalogue (ascending).
</ResponseField>

<ResponseField name="is_bestseller" type="boolean">
  Merchandising flag surfaced by the venue.
</ResponseField>

<ResponseField name="is_addon" type="boolean">
  If `true`, this treatment can only be added to another booking, not booked
  alone.
</ResponseField>

<ResponseField name="is_bundle" type="boolean">
  If `true`, this entry represents a treatment package / cure rather than a
  single service.
</ResponseField>

<ResponseField name="bundle_id" type="string | null">
  When `is_bundle` is `true`, the id of the underlying bundle definition.
</ResponseField>

<ResponseField name="available_days" type="integer[] | null">
  Days of the week the treatment is bookable, `0` (Sunday) … `6` (Saturday).
  `null` means no per-treatment restriction.
</ResponseField>

<ResponseField name="variants" type="Variant[]">
  List of bookable variants. May be empty if the treatment has no explicit
  variants.
</ResponseField>

## The Variant object

<ResponseField name="id" type="string">
  Variant identifier.
</ResponseField>

<ResponseField name="label" type="string">
  Variant label in French (e.g. `60 minutes`, `Duo`).
</ResponseField>

<ResponseField name="label_en" type="string | null">
  Variant label in English.
</ResponseField>

<ResponseField name="duration" type="integer">
  Duration in minutes for this variant.
</ResponseField>

<ResponseField name="price" type="number | null">
  Price as a decimal amount in the treatment's currency, or `null` if on
  request.
</ResponseField>

<ResponseField name="price_on_request" type="boolean">
  Whether this variant's price is on request.
</ResponseField>

<ResponseField name="guest_count" type="integer">
  Number of guests this variant is designed for (typically `1` or `2`).
</ResponseField>

<ResponseField name="is_default" type="boolean">
  Whether this is the variant shown by default.
</ResponseField>

<ResponseField name="sort_order" type="integer | null">
  Ordering hint among variants (ascending).
</ResponseField>
