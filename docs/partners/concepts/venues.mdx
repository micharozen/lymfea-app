---
title: "Venues"
description: "Spas and hotel spas connected to Eïa."
---

A **Venue** is the physical place where treatments are delivered. There are
two types of venues on Eïa:

- `hotel` — a spa inside a hotel. Guests can charge their booking to their
  room (the optional `guest.room_number` field applies).
- `spa` — an independent day spa. No room charge.

## The Venue object

<ResponseField name="id" type="string">
  Stable, opaque identifier for the venue.
</ResponseField>

<ResponseField name="slug" type="string">
  Human-readable identifier (e.g. `hotel-lutetia-paris`). Unique across Eïa.
</ResponseField>

<ResponseField name="type" type="'hotel' | 'spa'">
  Whether the venue is a hotel spa or a standalone day spa.
</ResponseField>

<ResponseField name="name" type="string">
  Display name in French.
</ResponseField>

<ResponseField name="name_en" type="string">
  Display name in English.
</ResponseField>

<ResponseField name="description" type="string | null">
  Long-form description in French.
</ResponseField>

<ResponseField name="description_en" type="string | null">
  Long-form description in English.
</ResponseField>

<ResponseField name="address" type="string">
  Street address.
</ResponseField>

<ResponseField name="city" type="string">
  City.
</ResponseField>

<ResponseField name="country" type="string">
  ISO 3166-1 alpha-2 country code (e.g. `FR`).
</ResponseField>

<ResponseField name="postal_code" type="string">
  Postal or ZIP code.
</ResponseField>

<ResponseField name="timezone" type="string">
  IANA timezone (e.g. `Europe/Paris`). Use this to render UTC timestamps in
  the venue's local time for your users.
</ResponseField>

<ResponseField name="currency" type="string">
  ISO 4217 currency code for pricing (e.g. `EUR`).
</ResponseField>

<ResponseField name="opening_time" type="string">
  Daily opening time in `HH:mm` format, in the venue's local timezone.
</ResponseField>

<ResponseField name="closing_time" type="string">
  Daily closing time in `HH:mm` format, in the venue's local timezone.
</ResponseField>

<ResponseField name="slot_interval" type="integer">
  Granularity of bookable slots, in minutes (typically `30`).
</ResponseField>

<ResponseField name="image" type="string | null">
  Primary image URL.
</ResponseField>

<ResponseField name="cover_image" type="string | null">
  Cover image URL, suitable for a hero banner.
</ResponseField>

## Example

```json
{
  "id": "hot_lutetia_paris",
  "slug": "hotel-lutetia-paris",
  "type": "hotel",
  "name": "Spa Akasha — Hôtel Lutetia",
  "name_en": "Akasha Spa — Hotel Lutetia",
  "description": "Un sanctuaire de bien-être au cœur du 6ᵉ arrondissement.",
  "description_en": "A wellness sanctuary in the heart of the 6th arrondissement.",
  "address": "45 Boulevard Raspail",
  "city": "Paris",
  "country": "FR",
  "postal_code": "75006",
  "timezone": "Europe/Paris",
  "currency": "EUR",
  "opening_time": "09:00",
  "closing_time": "21:00",
  "slot_interval": 30,
  "image": "https://cdn.lymfea.com/venues/hot_lutetia_paris/cover.jpg",
  "cover_image": "https://cdn.lymfea.com/venues/hot_lutetia_paris/hero.jpg"
}
```

<Note>
  Only venues with `status = 'active'` and that are in-scope for your API key
  are returned by `GET /v1/venues`. Inactive or archived venues are filtered
  out automatically.
</Note>
