---
title: "Retrieve a venue"
api: "GET /v1/venues/{venue_id}"
description: "Fetch a single venue's public profile."
---

Returns the public profile of one venue. Requires the `venues:read` scope.

<Note>
  This endpoint returns venue metadata only — it does **not** embed the
  treatment catalogue. To get the venue's treatments, call
  [`GET /v1/venues/{venue_id}/treatments`](/api-reference/venues/list-treatments).
</Note>

## Path parameters

<ParamField path="venue_id" type="string" required>
  The venue id.
</ParamField>

## Response

<ResponseField name="data" type="Venue">
  A [Venue](/concepts/venues) object (detail shape). The detail shape exposes
  scheduling fields (`opening_time`, `closing_time`, `slot_interval`,
  recurrence, booking-hold and out-of-hours settings) that the list shape
  omits.
</ResponseField>

## Errors

| HTTP | Body                              | Meaning                                            |
| ---- | --------------------------------- | -------------------------------------------------- |
| 403  | `{ "error": "Forbidden" }`        | Your key is scoped to another venue                |
| 404  | `{ "error": "Venue not found" }`  | Unknown id, or the venue is not public             |

## Example

<RequestExample>

```bash curl
curl https://api.saoma.io/v1/venues/8f1a9c3b-8d4e-4f21-a1b2-c3d4e5f60718 \
  -H "Authorization: Bearer $EIA_API_KEY"
```

```javascript Node.js
const res = await fetch(
  "https://api.saoma.io/v1/venues/8f1a9c3b-8d4e-4f21-a1b2-c3d4e5f60718",
  { headers: { Authorization: `Bearer ${process.env.EIA_API_KEY}` } }
);
const { data } = await res.json();
```

</RequestExample>
