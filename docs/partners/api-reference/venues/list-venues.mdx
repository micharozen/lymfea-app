---
title: "List venues"
api: "GET /v1/venues"
description: "List every active venue accessible to your API key."
---

Returns the active venues that are in scope for your API key. Requires the
`venues:read` scope.

<Note>
  If your API key is scoped to a single venue, this endpoint returns only that
  venue. Platform-wide keys receive every active venue, ordered by `name`.
</Note>

This endpoint takes **no query parameters** and is **not paginated** — it
returns the full in-scope list in one call. Cache the result; venue metadata
changes rarely.

## Response

<ResponseField name="data" type="Venue[]">
  Array of [Venue](/concepts/venues) objects (list shape — see the Venue
  concept for the exact fields).
</ResponseField>

## Example

<RequestExample>

```bash curl
curl https://api.saoma.io/v1/venues \
  -H "Authorization: Bearer $EIA_API_KEY"
```

```javascript Node.js
const res = await fetch("https://api.saoma.io/v1/venues", {
  headers: { Authorization: `Bearer ${process.env.EIA_API_KEY}` },
});
const { data } = await res.json();
```

```python Python
import os, requests

res = requests.get(
    "https://api.saoma.io/v1/venues",
    headers={"Authorization": f"Bearer {os.environ['EIA_API_KEY']}"},
)
data = res.json()["data"]
```
