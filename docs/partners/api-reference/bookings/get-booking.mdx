---
title: "Retrieve a booking"
api: "GET /v1/bookings/{booking_id}"
description: "Fetch the current state of a booking."
---

Returns the current state of a booking, including its status, assigned
therapist and scheduled time.

## Path parameters

<ParamField path="booking_id" type="string" required>
  The booking id returned by `POST /v1/bookings`. You can also pass your own
  `external_reference` by prefixing it with `ext:` (e.g.
  `ext:STAYCATION-ABC123`).
</ParamField>

## Response

Returns the [Booking](/concepts/bookings) object.

## Errors

| Code                    | HTTP | Meaning                                              |
| ----------------------- | ---- | ---------------------------------------------------- |
| `BOOKING_NOT_FOUND`     | 404  | Unknown booking id or external reference            |
| `FORBIDDEN`             | 403  | Booking exists but was not created by your API key  |

## Example

<RequestExample>

```bash curl
curl https://api-sandbox.lymfea.com/v1/bookings/bkg_7f1a9c3b8d4e4f21 \
  -H "Authorization: Bearer $LYMFEA_API_KEY"
```

```bash curl (by external_reference)
curl https://api-sandbox.lymfea.com/v1/bookings/ext:STAYCATION-ABC123 \
  -H "Authorization: Bearer $LYMFEA_API_KEY"
```

</RequestExample>
